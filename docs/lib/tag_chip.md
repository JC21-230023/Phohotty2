
# `lib/widgets/tag_chip.dart`

`tag_chip.dart`は、画像のタグを表示するために使用される、再利用可能なUIコンポーネントを定義します。

## 主な機能

- **タグの表示:**
  - `Chip`ウィジェットを使用して、タグのラベルを視覚的に魅力的な方法で表示します。
  - `label`プロパティにタグのテキストを設定し、`padding`と`backgroundColor`をカスタマイズして、一貫した外観を提供します。

- **再利用性:**
  - `StatelessWidget`として構築されており、タグの文字列をパラメータとして受け取るため、アプリケーション全体で簡単に再利用できます。
  - `TagLensPage`などのさまざまな画面で、画像のタグを表示するために使用されます。
