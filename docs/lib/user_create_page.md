# `pages/user_create_page.dart`

## 概要

このファイルは、新規ユーザー作成画面を提供します。

## 主な機能

*   メールアドレスとパスワードによるユーザー作成フォームを提供します。
    *   表示名、メールアドレス、パスワード、パスワード（確認）の入力フィールドがあります。
    *   入力値のバリデーション（入力必須、メール形式、パスワードの長さ、パスワードの一致）を行います。
    *   Firebase Authenticationを使用してユーザーを作成し、表示名を更新します。
*   Googleアカウントを使用したユーザー作成・サインイン機能を提供します。
    *   `FbAuth`サービスを使用してGoogleサインイン処理を呼び出します。
